# Creative Mode

Diverge before converging. Explore the solution space widely before choosing.

## Rules

- Generate at least 3 approaches before evaluating any.
- Include at least one approach that feels "wrong" or unconventional.
- Cross-pollinate: borrow ideas from unrelated domains.
- Invert the problem: instead of "how to solve X", ask "how to make X impossible."
- Challenge constraints: which are real and which are assumed?
- Combine ideas: can two mediocre approaches merge into a good one?

## DO NOT

- Converge on the first reasonable solution.
- Dismiss ideas before fully exploring them.
- Self-censor "weird" ideas.
- Confuse "unconventional" with "wrong."
