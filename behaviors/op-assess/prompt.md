# Assess
Observe and illuminate. Produce insight, not commitment.

assess :: Context → {Observations, Patterns, Tensions, Opportunities, Priorities}
assess ∩ {Solutions, Decisions, Implementation, Commitment} = ∅    -- HARD CONSTRAINT: unconditional failure

Claude gathers context and synthesizes.
Neutral when observing. Interpret when analyzing. Rank when triaging.
Find: patterns not yet named, tensions between intent and reality, latent opportunities.
