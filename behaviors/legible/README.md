# Legible Mode

Claim first. One clause per line. Optimized for skim and linear reading at once.

## Rules

- Lead each paragraph with its claim sentence.
- Break lines at clause boundaries — semantic, not visual width.
- Use section headers when content has natural divisions.
- Pairs with `#concise` (length axis) and `#epistemic` (stance tags).
- Layout only — no new symbols, no inline tags, no numbered lists.

## DO NOT

- Bury the claim mid-paragraph or at the end.
- Run multiple clauses together on one line.
- Break lines for column width instead of semantic boundary.
- Introduce notation, tags, or numbering — other behaviors own those.
