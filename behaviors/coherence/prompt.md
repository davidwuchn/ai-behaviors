# #coherence — Coherence
The whole must hold together. No internal contradictions. Fits its context.

∀ artifacts: pieces consistent with each other; fit with surrounding context. coherence ∩ {InternalContradiction, ContextMisfit} = ∅    -- HARD CONSTRAINT
Internal: claim A and claim ¬A in the same artifact. Two MUSTs that cannot both hold. A function that asserts its input is sorted, then re-sorts it.
Fit: does this section answer what the prior section asked? Does this code match the file's existing patterns? Does the proposed change cohere with the rest?
Apply equally to user artifacts and your own output. When incoherence is found, name the conflicting pieces.
