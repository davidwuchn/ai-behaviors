│ Snake Game - Implementation Plan                                             │
│                                                                              │
│ Context                                                                      │
│                                                                              │
│ Implement a Python3 + pygame snake game clone on an 8x8 board, launched via  │
│ a snake executable. The project lives in                                     │
│ /home/peter/git/ai-behaviors/output-examples/plan/.                      │
│                                                                              │
│ Files to Create                                                              │
│                                                                              │
│ 1. snake - Shell script launcher (~10 lines)                                 │
│ 2. snake_game.py - All game logic, single file (~250 lines)                  │
│                                                                              │
│ The venv/ directory is auto-created on first run.                            │
│                                                                              │
│ snake Launcher Script                                                        │
│                                                                              │
│ - Creates venv if missing, installs pygame                                   │
│ - Uses BASH_SOURCE[0] for path resolution so it works from any directory     │
│ - execs into the Python process for clean signal handling                    │
│ - chmod +x after creation                                                    │
│                                                                              │
│ snake_game.py Structure                                                      │
│                                                                              │
│ Constants                                                                    │
│                                                                              │
│ - GRID_SIZE=8, CELL_SIZE=64, HEADER_HEIGHT=48 (score/speed display area)     │
│ - Window: 512x560                                                            │
│ - BASE_TICK_RATE=5 (moves/sec at speed 1), TICK_INCREMENT=1 per speed level, │
│  MAX_TICK_RATE=15                                                            │
│ - STAR_SPAWN_CHANCE=0.05 (5% per tick when no star), STAR_LIFETIME=20 turns, │
│  STAR_GROWTH=3                                                               │
│ - POINTS_PER_SPEED=10                                                        │
│                                                                              │
│ Direction Model                                                              │
│                                                                              │
│ - Directions as (dx, dy) tuples: UP=(0,-1), DOWN=(0,1), LEFT=(-1,0),         │
│ RIGHT=(1,0)                                                                  │
│ - OPPOSITES dict for self-collision prevention                               │
│                                                                              │
│ Game Class                                                                   │
│                                                                              │
│ - State: snake (list of (x,y), head-first), direction, input_queue, score,   │
│ speed, alive, grow_pending, apple, star, star_timer                          │
│ - Snake starts: [(4,4), (3,4), (2,4)] moving RIGHT (size 3, center of board) │
│ - Growth: grow_pending counter; skip tail pop when > 0; apple adds 1, star   │
│ adds 3                                                                       │
│ - Key methods:                                                               │
│   - _spawn_apple() / _maybe_spawn_star() - place on random free cell         │
│   - _process_input_queue() - pop next valid direction, discard reversals     │
│   - tick() - process input, move head, check wall/self collision, check      │
│ food, star lifecycle                                                         │
│   - _update_speed() - speed = 1 + score // 10                                │
│   - get_tick_rate() - min(5 + (speed-1), 15)                                 │
│                                                                              │
│ Input Handling                                                               │
│                                                                              │
│ - KEY_MAP: WASD + arrow keys mapped to directions                            │
│ - Queue validated against last queued direction (not current direction) -    │
│ this makes rapid "wd" presses work correctly                                 │
│ - Enter triggers start/restart depending on state                            │
│                                                                              │
│ Rendering                                                                    │
│                                                                              │
│ - 60 FPS render loop, game logic at variable tick rate via accumulator       │
│ pattern                                                                      │
│ - Header shows score (left) and speed (right)                                │
│ - Snake: rounded rectangles (head brighter green, body darker green)         │
│ - Apple: red circle                                                          │
│ - Star: gold 8-point polygon                                                 │
│ - 2px cell inset so segments don't visually merge                            │
│ - Start/game-over: semi-transparent overlay with title + subtitle            │
│                                                                              │
│ Main Loop / State Machine                                                    │
│                                                                              │
│ - Three states: start_screen -> playing -> game_over -> playing              │
│ - Tick accumulator decouples 60 FPS rendering from variable game speed       │
│ - Multiple ticks can fire per frame at high speeds                           │
│                                                                              │
│ Self-Collision Details                                                       │
│                                                                              │
│ - Wall collision: head out of bounds = death                                 │
│ - Body collision: check against snake[:-1] if not growing, full snake if     │
│ growing (tail won't vacate)                                                  │
│ - Direction reversal: blocked in both direct input and queued input          │
│                                                                              │
│ Speed Progression                                                            │
│ ┌───────┬───────┬───────────┬──────────┐                                     │
│ │ Score │ Speed │ Tick rate │ Interval │                                     │
│ ├───────┼───────┼───────────┼──────────┤                                     │
│ │ 0-9   │ 1     │ 5/sec     │ 200ms    │                                     │
│ ├───────┼───────┼───────────┼──────────┤                                     │
│ │ 10-19 │ 2     │ 6/sec     │ 167ms    │                                     │
│ ├───────┼───────┼───────────┼──────────┤                                     │
│ │ 20-29 │ 3     │ 7/sec     │ 143ms    │                                     │
│ ├───────┼───────┼───────────┼──────────┤                                     │
│ │ 100+  │ 11    │ 15/sec    │ 67ms     │                                     │
│ └───────┴───────┴───────────┴──────────┘                                     │
│ Implementation Sequence                                                      │
│                                                                              │
│ 1. Create venv: python3 -m venv venv && venv/bin/pip install pygame          │
│ 2. Write snake_game.py (single file, all logic)                              │
│ 3. Write snake shell script                                                  │
│ 4. chmod +x snake                                                            │
│                                                                              │

│ - Confirm: start screen appears, enter starts game                           │
│ - Play: eat apple (score +1, snake grows by 1), verify speed change at 10    │
│ points                                                                       │
│ - Star: appears randomly, gold polygon, disappears after ~20 turns if        │

│ - Input queue: rapid "wd" turns up then right on consecutive ticks           │
│ - Self-collision: pressing opposite direction is ignored                     │
│ - Game over: wall/self collision shows overlay, enter restarts               │
